# Real-Time Fraud Detection Data Pipeline

An end-to-end data engineering and machine learning project for real-time fraud detection.

## Overview

This project simulates real-time financial transactions and processes them through a streaming data pipeline. The system is designed to detect potentially fraudulent transactions using a machine learning model, store transaction and prediction results in a database, and visualize fraud risks through an interactive dashboard.

## Tech Stack

- Python
- PostgreSQL
- Apache Kafka
- FastAPI
- Streamlit
- scikit-learn
- Docker

## Main Features

- Real-time transaction stream simulation
- Fraud detection with machine learning
- Transaction and prediction storage in PostgreSQL
- REST API for accessing prediction results
- Interactive dashboard for monitoring fraud risks
- Containerized development environment

## Dataset

The project is planned to use a financial transaction dataset such as PaySim for fraud detection experiments.
